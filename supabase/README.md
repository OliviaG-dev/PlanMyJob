# Supabase – Candidatures

## Créer la table

1. Ouvre ton projet sur [supabase.com](https://supabase.com) → **SQL Editor**.
2. Copie-colle le contenu de `migrations/001_create_candidatures.sql`.
3. Clique sur **Run** pour exécuter le script.

La table `candidatures` sera créée avec les colonnes et les politiques RLS (chaque utilisateur ne voit que ses propres candidatures).
